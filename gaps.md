## 详细说明

### 🔴 微信 H5 JS-SDK（核心裂变功能）

**为什么重要：** 龙虾学校的核心增长引擎是微信内分享。没有 JS-SDK 的分享功能，用户只能复制链接而无法自定义分享卡片（标题+描述+图片），裂变效率会大幅下降。

**应对方案：**
1. `skills/08-wechat-h5.md` 已提供完整的实现模板（前端+后端）
2. Agent 可以直接按模板编码，不需要外部 skill
3. 但必须等人类提供微信公众号 AppID/AppSecret 和配置 JS 安全域名

**降级方案：** 如果暂时拿不到微信公众号权限，先用浏览器原生 Web Share API 作为兜底，微信内用户仍可复制链接分享。

### 🟡 腾讯云 CVM 部署

**为什么不算严重：** 有 Docker skill + TAT 就能部署。不需要 SSH。

**应对方案：**
1. 方案 A：Docker build → TAT 远程执行 docker-compose up
2. 方案 B：用 CloudBase-MCP 一键部署（更简单但灵活性低）
3. 方案 C：前端用 Vercel，后端用腾讯云 CVM

### 🟡 前端埋点自动化

**为什么不算严重：** 埋点本身就是几行 JS 代码，不需要 skill 自动化。关键是知道**埋什么**，而不是**怎么埋**。

**应对方案：**
1. 参考 `skills/07-analytics.md` 中的必埋点列表
2. 用 PostHog JS SDK 手动集成（代码量极少）
3. 上线后用 GA4 MCP 分析数据
